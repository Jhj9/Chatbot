# Chatbot Frontend

1) git clone https://github.com/Jhj9/Chatbot.git<br>

2) mysite/secrets.json 생성 후 SECRET_KEY 입력 <br>
  Ex) {"SECRET_KEY": "django-insecure-dml=장고시크릿키50자"} <br>
  https://djecrety.ir/

3) python manage.py runserver<br>

4) http://127.0.0.1:8000/ 웹페이지 실행
